# Internal Package Documentation

This directory contains detailed implementation documentation for internal packages.

## Package Documentation

### Configuration (`config/`)
Documentation for the configuration system, including:
- Typed path system
- Validation performance
- Configuration suggestions
- Schema management

### GitOps (`gitops/`)
Documentation for GitOps repository generation:
- Template migration
- CLI integration
- Compatibility tracking
- Progress reporting
- Workspace implementation

### Services (`services/`)
Documentation for service management:
- Service lifecycle
- Plugin system
- Circular dependency detection
- Implementation summaries

### Template (`template/`)
Documentation for the template engine:
- Template composition
- Dependency resolution
- Embedded templates
- Feature flags
- Patch system
- Registry implementation
- Cache performance

### Testing (`testing/`)
Documentation for testing infrastructure and utilities.

### Talos (see `docs/providers/talos/`)
Talos provider implementation documentation has been moved to the providers section.

## Purpose

These documents provide:
- Implementation details for maintainers
- Architecture decisions and rationale
- Performance considerations
- Migration guides for internal changes
- Feature completion tracking

For user-facing documentation, see `docs/reference/` and `docs/overview.md`.
